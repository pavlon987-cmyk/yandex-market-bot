# 🤖 Yandex Market Trust Bot v3.0

Telegram бот для автоматической проверки траста аккаунтов Яндекс Маркета и парсинга заказов.

## 📋 Возможности

✅ Автоматическая проверка траста аккаунтов
✅ Парсинг истории заказов и возвратов
✅ Массовая загрузка аккаунтов из файла
✅ Детальная статистика по каждому аккаунту
✅ Экспорт данных в CSV
✅ Логирование всех действий
✅ Удобный Telegram интерфейс

## 🚀 Установка

1. **Клонируй репозиторий:**
```bash
git clone https://github.com/yourusername/yandex-market-bot.git
cd yandex-market-bot
```

2. **Установи зависимости:**
```bash
pip install -r requirements.txt
```

3. **Настрой конфиг:**
```bash
copy .env.example .env
notepad .env
